Treinamento do Modelo¶

A função de treinamento trainadaline() que recebe como entrada os dados e os parâmetros de treinamento e como saída fornece os pesos obtidos e o erro à cada época. O parâmetro de entrada par, quando em 1, indica que a entrada adicional +1 não foi acrescida aos dados e que ela deve ser adicionada dentro da função. A inicializaçãoo dos pesos então ocorre por meio de amostragem uniforme na faixa de valores -0.5 a +0.5 visando a dar chances iguais a pesos positivos e negativos durante o treinamento. Outra parte importante do código é a realização de sequência aleatória que visam a evitar sequências determinísticas de ajuste dos pesos. O restante do código envolve basicamente a aplicação da equação de ajuste dos pesos e o acúmulo do erro a cada iteração.

def trainadaline(xin, yd, eta, tol, maxepocas, par):
    dimxi = xin.shape
    N = dimxi[0]
    n = dimxi[1]

    if(par==1):
        wt = np.random.uniform(low=-0.5, high=0.5, size=n+1)
        xin = np.hstack((np.ones((N ,1)), xin))
    else:
        wt = np.random.uniform(low=-0.5, high=0.5, size=n)

    nepoca = 0
    eepoca = tol + 1

    evec = np.zeros((maxepocas, 1))
    while((nepoca < maxepocas) and (eepoca > tol)):
        ei2 = 0
        xseq = random.sample(range(N), N)
        for i in range(N):
            irand = xseq[i]
            ei = yd[irand] - np.dot(xin[irand,:], wt)
            dw = eta * ei * xin[irand,:]
            wt = wt + dw
            ei2 = ei2 + ei * ei
        evec[nepoca] =  ei2/N
        eepoca = evec[nepoca]

        nepoca += 1
    return np.reshape(wt,(wt.shape[0],1)), evec
